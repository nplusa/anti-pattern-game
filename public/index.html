How To Play

Players take turns placing X or O symbols. You lose if any pattern repeats 3 times consecutively.

Losing Example

X O X O O X O O X O O

XOO pattern repeated 3 times: XOO | XOO | XOO

Safe Example

X O X O X X O

XO appears 3 times but not consecutively - Safe!

Patterns can be any length. Stay alert!
